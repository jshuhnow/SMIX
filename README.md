# SC2ENV Multi-agent RL

## How to run
```buildoutcfg
$ virtualenv .env -ppython3
$ source .env/bin/activate
$ export SC2PATH=/your/sc2path
(.env) $ python3 run_model.py 
```
## Performance
With `num_runs=1000`, (winning rate)=99.8%.
